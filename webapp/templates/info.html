All Bot Functions

Music Commands:

clear Purges queue

leave Leaves voice.

nowplaying Embed of song

pause Pauses song.

play Pass song or query

playlist Pass in a playlist URL

queue Displays queue

repeat Repeats current song, once.

repeat3 Repeat current song 3x.

repeat5 Repeat current song 5x

resume Resume, if paused.

skip Skips current song

join if the bot says you need to be in a voice channel use this


Basic Commands:

wisdom A nugget of wisdom

choose if you can't decide between two things, use this command

cool are you cool

delete deletes a specified amount of messages from a channel

delrr removes the role and emoji from react role list

dm dms a user from the bot

help Shows this message

leave1 Second leave command if the first one doesnt work

level check your level

newrc adds a new role command

newrr allows a role to be assigned with an emoji

newrrmsg sets the message you react to to get roles

pokeman Generates a random pokemon

poll create a poll for important desicions

r repeats up to 29 times

se find top ten results on google for the message

se1 finds one result on google for the message

status Changes the bots status
